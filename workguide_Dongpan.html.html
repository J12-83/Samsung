시작 전 — 동판이란?

동판은 박(호일) · 형압(엠보싱/디보싱) 인쇄에 사용하는 금속 원판입니다.
파일을 잘못 만들면 동판 자체가 잘못 제작됩니다. 아래 순서를 반드시 지켜주세요.

STEP 1

동판 소재 선택

생산수량에 따라 소재가 달라집니다.

용도소재
샘플 · 1회성 · 소량 생산아연판
반복 · 대량 생산구리판
STEP 2

형압 종류 확인 (엠보싱 / 디보싱)

형압 동판을 주문할 경우, 양각/음각을 반드시 구분해야 합니다.

종류별칭효과
양각형압엠보싱볼록
음각형압디보싱오목
💡 엠보싱 = 볼록, 디보싱 = 오목
⚠️ 인쇄물 결과 기준입니다.
STEP 3

컬러 모드를 CMYK로 설정

인쇄용 파일은 반드시 CMYK 모드여야 합니다.
RGB로 작업하면 색상이 달라질 수 있습니다.

▲ 타이틀 바에 (CMYK) 표시 확인
설정 방법
  1. 파일(File) → 문서 색상 모드 → CMYK 색상 클릭
  2. 타이틀 바에 (CMYK) 표시 확인
⚠️ (RGB) 표시 시 반드시 CMYK로 변경
💡 가는 선 두께의 경우, 0.2mm 이상을 추천합니다. (너무 얇은 글씨나 선은 박이 뜯기거나 번져 보일 수 있습니다.)
STEP 4

K=100 (순수 검정) 설정

동판은 K=100 기준으로 제작됩니다.

주문 유형로고배경
박 / 디보싱 K=100 K=0
엠보싱 K=0 K=100
✅ 박 / 디보싱 예시
✅ 엠보싱 예시
K=100 설정 방법
  1. 윈도우(Window) → 색상(Color) 패널 열기
  2. C=0 / M=0 / Y=0 / K=100 입력
STEP 5

이미지 데이터 처리

※ 벡터 데이터가 기본입니다.
※ 부득이하게 이미지 사용 시 아래를 따릅니다.
5-1. 이미지 색상을 K=100으로 변환
변환 방법
  1. 이미지 선택 → 색상 패널에서 K=100 설정
5-2. 이미지 임베드(포함) 처리
임베드 방법
  1. 윈도우(Window) → 링크(Links) 패널
  2. 이미지 선택 → 메뉴(≡) → 이미지 포함
  3. 🔗 아이콘 사라지면 완료
❌ 임베드 전 — 이미지 연결 상태
✅ 임베드 후 — 이미지 포함 상태
⚠️ 저화질 이미지는 동판에서 픽셀이 지글거려 보일 수 있습니다.
⚠️ 임베드하지 않고 저장하면 이미지가 누락될 수 있습니다.
STEP 6

불투명도 확인

오브젝트가 반투명하면 동판 제작에 오류가 생길 수 있습니다.

▲ 불투명도 100% / 표준(Normal)
확인 방법
  1. Ctrl+A → 전체 선택
  2. 윈도우 → 투명도(Transparency)
  3. 불투명도 100% / 표준(Normal)
STEP 7

텍스트 아웃라인 처리

텍스트를 패스(도형)로 변환합니다.
폰트 없는 PC에서 글자 깨짐을 방지합니다.

처리 방법
  1. Ctrl+A → 전체 선택
  2. 문자(Type) → 윤곽선 만들기  Ctrl+Shift+O
  3. 앵커 포인트 나타나면 완료
⚠️ 아웃라인 후 텍스트 수정 불가. 최종 확인 후 진행
STEP 8

비인쇄 영역 K=0 설정

가이드 라인, 메모 등에 K값이 있으면 동판에 표현됩니다.

처리 방법
  1. 비인쇄 오브젝트 선택
  2. 윈도우 → 색상(Color) 에서 K=0 설정
⚠️ 미세한 K값도 동판에 표현됩니다. 반드시 확인
STEP 9

불필요한 레이어 삭제

숨겨진/빈 레이어가 남아 있으면 오류 원인이 됩니다.

▲ 숨김/빈 레이어 확인 후 삭제
처리 방법
  1. 윈도우 → 레이어(Layers) 패널
  2. 숨김/빈 레이어 확인
  3. 선택 → 휴지통 아이콘 클릭 삭제
